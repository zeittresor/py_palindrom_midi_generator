# py_palindrom_midi_generator
Creates a midi classical palindrom orchester song part by simply pressing a single button.

<img width="567" height="183" alt="palindrom_generator_img" src="https://github.com/user-attachments/assets/362b7a90-ceb9-48f8-b30a-76539cf68ac8" />

Output .mid files will be created in the same folder where the script is.

Requires: pip install mido
